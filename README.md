# Telegram Video Downloader Bot

بوت تيليغرام لتحميل الفيديوهات من أي منصة (يوتيوب، إنستغرام، تيك توك، تويتر...)

## المتطلبات

- Python 3.11+
- `pip install -r requirements.txt`

## المتغيرات البيئية

```
TELEGRAM_BOT_TOKEN=توكن_البوت
CHANNEL_ID=معرف_القناة (اختياري)
CHANNEL_INVITE=رابط_دعوة_القناة (اختياري)
```

## تشغيل البوت

```bash
python bot.py
```
